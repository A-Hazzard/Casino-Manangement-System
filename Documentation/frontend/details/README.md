# Detail Pages Documentation

This directory contains documentation for detail pages that provide in-depth views of specific resources.

## Pages in This Directory

- **[Location Details](location-details.md)** - Detailed location information, metrics, and management
- **[Machine Details](machine-details.md)** - Individual machine configuration, metrics, and SMIB settings
- **[Collection Report Details](collection-report-details.md)** - Detailed collection report analysis with variance comparison
- **[Location Machines](location-machines.md)** - Location-machine relationship management and assignments

## Documentation Structure

Each detail page documentation includes:

1. **Overview** - Page purpose and key features
2. **File Information** - Source files and component structure
3. **Page Sections** - Detailed UI breakdown
4. **API Endpoints** - Backend integration points
5. **State Management** - Data flow and state handling
6. **Key Functions** - Important business logic

## Common Features

Detail pages typically include:

- **Resource Information** - Complete resource details
- **Related Data** - Associated resources and relationships
- **Metrics and Analytics** - Performance data and statistics
- **Action Buttons** - Edit, delete, and management actions
- **Navigation** - Breadcrumbs and related links
